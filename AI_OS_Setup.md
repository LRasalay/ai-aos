# AI Operating System Setup

This template contains:
- Central AI prompts (core, tasks, profiles)
- Tool wrappers for `.claude` and `.windsurf`
- Workflows ready to run with ai-cli

## After downloading:
1. Initialize Git
2. Add ai-prompts submodule
3. Create symlinks in `.claude` & `.windsurf`
4. Use `./ai-cli run <task> <tool> "<description>"`
